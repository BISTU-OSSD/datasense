# DataSense Web - 本地网页端数据分析工具

这是一个基于Flask的Web应用，将DataSense的命令行功能整合到网页界面。

## 功能特性

- ✅ 上传CSV文件进行分析
- ✅ 一键加载示例数据集（鸢尾花）
- ✅ 数据概览：行列数、列信息、缺失值统计
- ✅ 统计分析：均值、标准差、四分位数等
- ✅ 数据预览：查看前10行数据
- ✅ 数据可视化：散点图、直方图
- ✅ 美观的响应式UI界面

## 快速启动

### 1. 安装依赖

```bash
cd web
pip install -r requirements.txt
```

### 2. 启动Web应用

```bash
python app.py
```

### 3. 访问网页

打开浏览器，访问：**http://localhost:5000**

## 使用说明

### 方法一：上传自己的数据
1. 点击上传区域或拖拽CSV文件
2. 等待数据加载完成
3. 查看数据概览和统计信息
4. 在可视化区域选择列并生成图表

### 方法二：使用示例数据
1. 点击"加载示例数据集"按钮
2. 系统自动加载鸢尾花数据集
3. 直接查看分析结果和图表

## 界面说明

### 数据加载区
- 支持拖拽上传或点击选择
- 支持加载预置的示例数据集

### 数据概览
- 显示文件名、行列数
- 显示每列的数据类型和缺失值情况

### 统计分析
- 为每个数值列显示统计指标
- 包括：均值、标准差、最小值、四分位数、最大值

### 数据预览
- 显示前10行数据
- 方便快速了解数据内容

### 数据可视化
- 散点图：查看两个变量的关系
- 直方图：查看单个变量的分布

## 注意事项

1. 首次运行需要安装依赖
2. 上传文件大小限制为16MB
3. 仅支持CSV格式文件
4. 图表会实时生成并显示在页面上

## 常见问题

### Q: 为什么上传文件后没有反应？
A: 请检查文件是否为CSV格式，且文件大小不超过16MB。

### Q: 如何修改端口号？
A: 编辑 `app.py` 文件最后一行的 `port=5000` 即可。

### Q: 可以在局域网内访问吗？
A: 可以，应用默认监听 `0.0.0.0`，局域网内其他设备可通过你的IP访问。
